# إت ماستر · ET MASTER — موقع المطعم

موقع تعريفي فاخر (Arabic RTL، vanilla static) لمطعم **إت ماستر** للحوم والستيك في بريدة، القصيم.
ستيك معتّق تركي ستايل، مشاوي لحم، برغر ومقبّلات.

## الرابط المباشر
`https://az212z.github.io/et-master/` (بعد النشر على GitHub Pages)

## الحالة التقنية (Backend)
- موقع **ثابت بالكامل (static)** — لا يوجد سيرفر أو قاعدة بيانات.
- نموذج الحجز **تجريبي للعرض**: يحفظ نسخة محليًّا في `localStorage` ثم يفتح **واتساب** برسالة عربية جاهزة إلى `966502225533`. لا يُرسل بريدًا ولا يخزّن بيانات على خادم.
- أزرار التواصل العائمة: واتساب + اتصال + خرائط قوقل.

## مصدر الصور
الصور **حقيقية للمطعم** مأخوذة من ملفّه على خرائط قوقل، وتمّت **معالجتها ومعايرة ألوانها** (color-graded) مسبقًا قبل الاستخدام. تمّت تنقية الصور: استُبعدت لوحات الأسعار والصور التي تحتوي وجوهًا.

## الموشن التوقيعي
- **Hero — كشف اللحم المعتّق (Dry-Age Reveal):** قطعة لحم معلّقة مُضاءة بالسبوت، يهوي عليها ساطور فيُفتح خط قطع يكشف الداخل الأحمر المعرّق + نفخة بخار + توهّج جمر + لمعة فولاذية.
- **طبقة حياة محيطة:** بارالاكس خفيف على الهيرو (مؤشر + سكول، ≤12px، معطّل على اللمس)، جزيئات جمر تصعد، توهّج "يتنفّس".
- **كوريغرافيا سكول:** ظهور اتجاهي زنبركي متدرّج، رسم مونوغرام SVG عند التحميل، count-up للتقييم.
- كل الحركة تُلغى وتستقرّ على الحالة النهائية عند `prefers-reduced-motion`.
